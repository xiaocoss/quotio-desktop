# 粉色花卉主题素材

本目录是 `粉色花卉主题` 的完整设计素材包，所有文件均按用途分类，避免与 `设计图/原版/` 混放。

## 目录结构

- `人物/`：独立人物头像和人物主视觉原图。
- `花卉/`：透明玫瑰角饰和透明花瓣叠层。
- `云纹/`：透明可缩放的浅金云纹、卷草纹、九宫格边框与使用说明。
- `背景/`：柔和纸感主题背景原图。
- `空状态/`：透明接口/服务空状态插画。

程序运行时请使用 `apps/desktop/public/rose/` 中经过尺寸和格式优化的对应文件；这里的文件用于设计留档、二次编辑和重新导出。

## 页面对应

- 仪表盘：人物、花卉、背景、页面角云纹、分段括号、枝叶卷草。
- 额度：花卉、背景、不对称面板云纹框。
- 智能体：背景、页面角云纹、小卡片回勾纹、头像花环。
- 日志与代理日志：背景、花卉、页面角云纹、侧栏 T 形云结。
- 设置：背景、花卉、页面角云纹。
- 关于：人物、花卉、背景、不对称面板云纹框、枝叶卷草和头像花环。
- 服务商、2FA、悬浮窗：使用花卉、背景或空状态素材，不使用云纹套件。
