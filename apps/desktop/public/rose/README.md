# Rose theme runtime assets

These files are project-bound runtime assets for the selectable `rose` theme.

| File | Alpha | Suggested use |
| --- | --- | --- |
| `character-avatar.png` | No | Sidebar/profile/avatar image |
| `sidebar-rose-logo-v2.png` | Yes | Generated watercolor rose emblem for the sidebar wordmark |
| `sidebar-footer-floral-v1.png` | Yes | Generated watercolor floral layer behind the sidebar proxy card |
| `character-hero.webp` | No | Dashboard and About hero artwork; blends into the rose page background |
| `character-dashboard-hero-cutout-v4.png` | Yes | Transparent dashboard character cutout; layer above the top-right flowers |
| `about-character-cutout-v1.png` | Yes | Generated About-page portrait cutout matching the floral hero composition |
| `rose-corner.png` | Yes | Corner decoration; rotate/mirror in CSS |
| `rose-petals.png` | Yes | Sparse overlay with `pointer-events: none` |
| `empty-endpoints.png` | Yes | Providers empty state |
| `rose-bg.webp` | No | App/page background texture |
| `rose-mark.svg` | Vector | Rose icon; color through `currentColor` |
| `cloud-filigree-corner.svg` | Vector | Main content/page corner; mirror for other corners |
| `cloud-dashboard-corner.svg` | Vector | Square dashboard-frame corner cloud; rotate for all four corners |
| `cloud-panel-9slice.svg` | Vector | Asymmetric ornamental panel frame via `border-image` |
| `cloud-card-9slice.svg` | Vector | Small hooked card corners via `border-image` |
| `cloud-rail-junction.svg` | Vector | Log-page sidebar/content T junction; rotate for bottom |
| `cloud-segment-bracket.svg` | Vector | Dashboard segmented-control side bracket |
| `cloud-botanical-sprig.svg` | Vector | Character-side botanical line ornament |
| `cloud-avatar-wreath.svg` | Vector | Avatar/portrait wreath backdrop |

Do not extract UI assets from the full-page mockups. Use the files in this directory directly.

Detailed cloud/filigree usage is documented in `cloud-assets.md`.
