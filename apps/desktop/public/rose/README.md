# Rose theme runtime assets

These files are project-bound runtime assets for the selectable `rose` theme.

| File | Alpha | Suggested use |
| --- | --- | --- |
| `character-avatar.png` | No | Sidebar/profile/avatar image |
| `character-hero.webp` | No | Dashboard and About hero artwork; blends into the rose page background |
| `rose-corner.png` | Yes | Corner decoration; rotate/mirror in CSS |
| `rose-petals.png` | Yes | Sparse overlay with `pointer-events: none` |
| `empty-endpoints.png` | Yes | Providers empty state |
| `rose-bg.webp` | No | App/page background texture |
| `rose-mark.svg` | Vector | Rose icon; color through `currentColor` |

Do not extract UI assets from the full-page mockups. Use the files in this directory directly.
