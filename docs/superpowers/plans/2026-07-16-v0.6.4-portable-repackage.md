# Quotio v0.6.4 Portable Repackage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reassemble the existing Quotio v0.6.4 Windows executable into a verified no-install directory and ZIP without recompiling or changing source files.

**Architecture:** Use `scripts/package-portable.ps1` as the single assembly boundary. It copies the existing `target/release/quotio-desktop.exe`, icon, repository resource notes, and the three locally cached offline components into a fresh versioned directory, then creates a ZIP. Verification compares the packaged executable hash with the source artifact and checks required archive entries.

**Tech Stack:** PowerShell 7, npm script runner, Tauri 2 release artifact, `System.IO.Compression`, SHA-256 file hashes.

### Task 3: Verify contents, executable identity, and handoff metadata

**Files:**
- Read: `dist-portable/Quotio_0.6.4_x64_portable/`
- Read: `dist-portable/Quotio_0.6.4_x64_portable.zip`

- [ ] **Step 1: Verify required files in the directory and ZIP**

Run:

```powershell
$dir = 'dist-portable/Quotio_0.6.4_x64_portable'
$zip = "$dir.zip"
$required = @(
  'Quotio.exe',
  'resources/proxy/windows/CLIProxyAPI.exe',
  'resources/kiro/windows/kiro-rs.exe',
  'resources/proxy/windows/plugins/quotio-key-router.dll'
)
Add-Type -AssemblyName System.IO.Compression.FileSystem
$archive = [IO.Compression.ZipFile]::OpenRead((Resolve-Path $zip))
try {
  foreach ($relative in $required) {
    if (-not (Test-Path (Join-Path $dir $relative) -PathType Leaf)) { throw "Missing directory file: $relative" }
    if (-not $archive.GetEntry(($dir.Split('/')[-1] + '/' + $relative))) { throw "Missing ZIP entry: $relative" }
  }
} finally {
  $archive.Dispose()
}
Write-Output 'content-check: required files present in directory and ZIP'
```

Expected: `content-check: required files present in directory and ZIP`.

- [ ] **Step 2: Compare the packaged executable SHA-256**

Run:

```powershell
$sourceHash = (Get-FileHash 'target/release/quotio-desktop.exe' -Algorithm SHA256).Hash
$portableHash = (Get-FileHash 'dist-portable/Quotio_0.6.4_x64_portable/Quotio.exe' -Algorithm SHA256).Hash
if ($sourceHash -ne $portableHash) { throw "Executable hash mismatch: $sourceHash vs $portableHash" }
$zipHash = (Get-FileHash 'dist-portable/Quotio_0.6.4_x64_portable.zip' -Algorithm SHA256).Hash
$zipSize = [math]::Round((Get-Item 'dist-portable/Quotio_0.6.4_x64_portable.zip').Length / 1MB, 2)
Write-Output "exe-sha256: $portableHash"
Write-Output "zip-size-mib: $zipSize"
Write-Output "zip-sha256: $zipHash"
```

Expected: no mismatch error; the executable hash equals the existing v0.6.4 artifact hash, and the ZIP size is greater than zero.
